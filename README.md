# Web Teknolojileri Dersi Proje Ödevi

Sakarya Üniversitesi Bilgisayar Mühendisliği bölümünde aldığım Web Teknolojileri dersinin dönem projesi. Ödevi yaparken izlediğim süreci ve her gün yaptığım push işlemlerini bu repo üzerinden takip edebilirsiniz.

## Bilgilerim
* Ad Soyad: Tuna Karadeniz
* Bölüm: Bilgisayar Mühendisliği
* Numara: B241210019
* Ders: Web Teknolojileri

## Neler Yapacağım?
Ödev kapsamında; semantik HTML5 yapısında bir CV sayfası, responsive bir şehir tanıtım sayfası, dışarıdan veri çeken bir API sayfası ve PHP ile çalışan güvenli bir login sistemi hazırlamam gerekiyor. Tasarımda Bootstrap kullanacağım ve form kontrollerini hem JS hem de bir framework (Vue.js gibi) üzerinden çift taraflı yapacağım.
