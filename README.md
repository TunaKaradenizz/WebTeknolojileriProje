# Web Teknolojileri Dersi Proje Ödevi

Sakarya Üniversitesi Bilgisayar Mühendisliği bölümünde aldığım Web Teknolojileri dersinin dönem projesi. Ödevi yaparken izlediğim süreci ve her gün yaptığım push işlemlerini bu repo üzerinden takip edebilirsiniz.

## Bilgilerim
* Ad Soyad: Tuna Karadeniz
* Bölüm: Bilgisayar Mühendisliği
* Numara: B241210019
* Ders: Web Teknolojileri

## Neler Yapacağım?
Ödev kapsamında; semantik HTML5 yapısında bir CV sayfası, responsive bir şehir tanıtım sayfası, dışarıdan veri çeken bir API sayfası ve PHP ile çalışan güvenli bir login sistemi hazırlamam gerekiyor. Tasarımda Bootstrap kullanacağım ve form kontrollerini hem JS hem de bir framework (Vue.js gibi) üzerinden çift taraflı yapacağım.

## Çalışma Planım (10 Gün)
Hocanın istediği 10 farklı günde push kuralına uymak için şöyle bir plan yaptım. İşler bittikçe buradaki maddelerin üstünü çizerek ilerleyeceğim.

* ~~**Gün 1:** Repo kurulumu, klasörlerin ayarlanması ve README/gitignore dosyalarının halledilmesi.~~

* **Gün 2:** Hakkında sayfası içeriğinin ve medya öğelerinin hazırlanması.

* **Gün 3:** Özgeçmiş sayfası kodlaması (header, section gibi semantik taglar kullanılacak).

* **Gün 4:** Şehrim sayfası ve içindeki 4 resimli slider yapısı.

* **Gün 5:** Mirasımız veya Takımımız bölümünün detaylandırılması.

* **Gün 6:** Bir API servisinden veri çekip sitede gösterme işlemi.

* **Gün 7:** İletişim sayfasındaki formun tüm elemanlarla beraber tasarlanması.

* **Gün 8:** Form denetimlerinin hem düz JS hem de Vue.js/React ile kodlanması.

* **Gün 9:** Login sayfası tasarımı ve PHP tarafındaki şifre kontrol mekanizması.

* **Gün 10:** Form verilerinin PHP ile ekrana yazdırılması, son kontroller ve hostinge yükleme.
